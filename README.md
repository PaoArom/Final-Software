# Final-Software

Project Name: 
    Educational escape room game.

Description: 
    The program is an escape room game on console. It is designed to teach players how to respond to real-life dangerous situations, such as fire, earthquakes, medical emergencies, or natural disasters. The player is in a simulated emergency scenario and must make decisions, solve puzzles, and manage time to escape safely. Wrong decisions and running out of time both result in losing the game. The goal is to prepare the players for emergencies and make safety awareness engaging and interactive by combining it with game mechanics that challenge critical thinking and decision-making under pressure.

Core features:
    - The game will have a login to save players' progress.
    - The game will have multiple endings based on player decision.
    - The game will have a time system, if the player doesn't beat the level on time, then is game over.
    - The game will have multiple levels.
    - The game will have a tutorial level that teacher the player how the mechanics work.
